Fonte variável Figtree
=====================

Este download contém Figtree como fontes variáveis ​​​​e estáticas.

Figtree é uma fonte variável com este eixo:
  o que

Isso significa que todos os estilos estão contidos nestes arquivos:
  Figtree-VariableFont_wght.ttf
  Figtree-Italic-VariableFont_wght.ttf

Se o seu aplicativo oferece suporte total a fontes variáveis, agora você pode escolher estilos intermediários
que não estão disponíveis como fontes estáticas. Nem todos os aplicativos suportam fontes variáveis ​​e
nesses casos você pode usar os arquivos de fontes estáticas para Figtree:
  estático/Figtree-Light.ttf
  estático/Figtree-Regular.ttf
  estático/Figtree-Medium.ttf
  estático/Figtree-SemiBold.ttf
  estático/Figtree-Bold.ttf
  estático/Figtree-ExtraBold.ttf
  estático/Figtree-Black.ttf
  estático/Figtree-LightItalic.ttf
  estático/Figtree-Itálico.ttf
  estático/Figtree-MediumItalic.ttf
  estático/Figtree-SemiBoldItalic.ttf
  estático/Figtree-BoldItalic.ttf
  estático/Figtree-ExtraBoldItalic.ttf
  estático/Figtree-BlackItalic.ttf

iniciar
-----------

1. Instale os arquivos de fonte que deseja usar

2. Use o seletor de fontes do seu aplicativo para visualizar a família de fontes e todas as
estilos disponíveis
